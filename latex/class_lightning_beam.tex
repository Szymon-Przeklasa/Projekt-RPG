\doxysubsection{Detailed Description}
Klasa odpowiedzialna za wizualny efekt błyskawicy. Generuje dynamiczną, nieregularną linię pomiędzy dwoma punktami, imitującą elektryczny łuk. 

Efekt:\+ 
\begin{DoxyItemize}
\item tworzy poszarpaną linię między źródłem a celem, 
\item losowo zmienia grubość linii dla efektu migotania, 
\item automatycznie usuwa się po zakończeniu animacji. 
\end{DoxyItemize}
