\doxysubsection{Detailed Description}
Główna klasa sceny gry, zarządzająca odliczaniem czasu rundy i warunkiem zwycięstwa. Timer jest uruchamiany automatycznie po załadowaniu sceny i zatrzymywany po pauzie lub śmierci gracza. Po osiągnięciu czasu Win\+Time wywoływana jest metoda On\+Win, która dezaktywuje spawner wrogów i wyświetla ekran wyników (\doxylink{class_win_screen}{Win\+Screen}).
