\doxysubsection{Detailed Description}
Interfejs użytkownika odpowiedzialny za wybór ulepszenia po awansie gracza. 

Po osiągnięciu nowego poziomu:\+ 
\begin{DoxyItemize}
\item zatrzymuje rozgrywkę, 
\item losuje maksymalnie 3 dostępne ulepszenia, 
\item wyświetla je jako przyciski wyboru, 
\item po wyborze stosuje ulepszenie i zamyka panel. 
\end{DoxyItemize}
