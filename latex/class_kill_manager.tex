\doxysubsection{Detailed Description}
Menedżer odpowiedzialny za zliczanie zabójstw przeciwników. Implementuje wzorzec Singleton i zapisuje dane do pliku. 

Przechowuje dwa typy danych:\+ 
\begin{DoxyItemize}
\item {\bfseries{kills}} — całkowita liczba zabójstw (zapisywana między sesjami), 
\item {\bfseries{session\+Kills}} — zabójstwa tylko z bieżącej rozgrywki. 
\end{DoxyItemize}
