\doxysubsection{Detailed Description}
Interfejs menu pauzy wywoływany klawiszem ESC. 

Menu:\+ 
\begin{DoxyItemize}
\item zatrzymuje rozgrywkę, 
\item blokuje działanie broni gracza, 
\item umożliwia restart poziomu, 
\item pozwala wyświetlić statystyki i autorów, 
\item umożliwia powrót do menu głównego. 
\end{DoxyItemize}
