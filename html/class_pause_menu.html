Detailed Description

Interfejs menu pauzy wywoływany klawiszem ESC.

Menu:

  • zatrzymuje rozgrywkę,
  • blokuje działanie broni gracza,
  • umożliwia restart poziomu,
  • pozwala wyświetlić statystyki i autorów,
  • umożliwia powrót do menu głównego.
