Detailed Description

Reprezentuje dane ulepszenia (broń, pasywka lub statystyka).

Każde ulepszenie składa się z wielu poziomów, z których każdy posiada własny opis i efekt.

System działa w stylu Vampire Survivors: gracz wybiera ulepszenie, a system automatycznie aplikuje odpowiedni poziom.
