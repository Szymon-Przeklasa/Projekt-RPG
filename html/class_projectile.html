Detailed Description

Klasa bazowa pocisku w grze. Odpowiada za ruch, kolizję oraz zadawanie obrażeń przeciwnikom.

virtual void Projectile.OnHit ( Node body)
protectedvirtual

Obsługuje trafienie w obiekt. Jeśli trafiony obiekt to przeciwnik:

  • zadaje obrażenia,
  • aplikuje knockback,
  • zmniejsza liczbę przebić (pierce),
  • usuwa pocisk, jeśli nie ma już przebić.
