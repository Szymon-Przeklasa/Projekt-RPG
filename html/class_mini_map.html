Detailed Description

Miniatura mapy wyświetlana w UI gry jako SubViewport. Śledzi pozycję gracza przez przesuwanie wewnętrznej kamery. Używa tego samego World2D co główna scena, dzięki czemu renderuje obiekty ze sceny gry bez ich duplikowania.
